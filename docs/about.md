## О проекте

Этот проект был создан для демонстрации возможностей создания кастомных тем для MkDocs с использованием:

1. **Jinja2 шаблонов** для кастомной верстки
2. **CSS** для стилизации
3. **GitHub Actions** для автоматического деплоя

### Особенности реализации

#### Кастомная тема
- Переопределены базовые шаблоны MkDocs
- Созданы кастомные header и footer
- Добавлены метаданные для SEO

#### Стилизация
- Современный CSS с переменными
- Адаптивный дизайн
- Анимации и переходы

#### Автоматизация
- CI/CD пайплайн с GitHub Actions
- Автоматическая сборка и деплой
- Кэширование зависимостей
